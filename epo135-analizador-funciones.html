Analizador de Funciones Matemáticas

Herramienta interactiva para estudiantes de preparatoria. Explora funciones, manipula parámetros en tiempo real, analiza sistemas de ecuaciones y exporta tus gráficas.

Institución: EPO 135
Nivel: Bachillerato general
Versión: 3.0 — Marzo 2026
Modo: Offline / Sin internet
Módulos disponibles
📈
11 tipos

Análisis de funciones

Graficador interactivo con sliders para cada parámetro. Encuentra raíces, tabla de valores y exporta imagen.

⚖️
7 tipos

Sistemas de ecuaciones

Lineales 2×2 y 3×3, y sistemas mixtos (lineal + cuadrática, cúbica, raíz, exponencial, logarítmica).

📐

Funciones trigonométricas

Seno, coseno y tangente con control de amplitud, frecuencia, desfase y desplazamiento vertical.

📊

Tabla de valores

Genera tablas de 21 puntos en el rango visible. Las raíces se marcan automáticamente en azul.

🎯

Búsqueda de raíces

Algoritmo de bisección con precisión de 4 decimales. Las raíces aparecen como letreros sobre la gráfica.

🖼️

Exportar imagen

Descarga la gráfica como PNG con marca institucional EPO 135 para imprimir o compartir.

Mar 2026

Versión actual — Sistema completo

  • Pestaña "Sistemas de ecuaciones": lineal 2×2 y 3×3 (Gauss-Jordan), lineal+cuadrática, lineal+cúbica, lineal+raíz, lineal+exponencial, lineal+logarítmica
  • Intersecciones graficadas con letreros de coordenadas (algoritmo bisección)
  • Diagnóstico de sistemas: único, infinito, incompatible con badges de color
  • Eliminación de cotangente, secante y cosecante
  • Nuevas funciones: logaritmo base b (slider para la base) y logaritmo natural ln
  • Dashboard de bienvenida con changelog y módulos
  • Archivo HTML standalone completo para uso offline
